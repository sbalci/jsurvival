# large_performance_data: example dataset for the jsurvival module

Example dataset distributed with the jsurvival jamovi module for
demonstration and testing.

## Usage

``` r
large_performance_data
```

## Format

A data frame with 2000 rows and 9 variables:

- patient_id:

  integer variable

- age:

  numeric variable

- sex:

  factor variable

- cancer_type:

  factor variable

- treatment_center:

  factor variable

- old_stage:

  factor variable

- new_stage:

  factor variable

- survival_time:

  numeric variable

- event:

  integer variable
