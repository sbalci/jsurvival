# Calculate F1 Score (Harmonic Mean of Precision and Recall)

Calculate F1 Score (Harmonic Mean of Precision and Recall)

## Usage

``` r
.calculateF1Score(tp, fp, fn)
```

## Arguments

- tp:

  Number of true positives

- fp:

  Number of false positives

- fn:

  Number of false negatives

## Value

F1 score (0-1)
