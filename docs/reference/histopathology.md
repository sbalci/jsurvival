# histopathology: example dataset for the jsurvival module

Example dataset distributed with the jsurvival jamovi module for
demonstration and testing.

## Usage

``` r
histopathology
```

## Format

A data frame with 250 rows and 38 variables:

  numeric variable

- Name:

  character variable

  character variable

- Age:

  numeric variable

  character variable

  character variable
