# Calculate Positive Predictive Value (Precision)

Calculate Positive Predictive Value (Precision)

## Usage

``` r
.calculatePPV(tp, fp, prevalence = NULL)
```

## Arguments

- tp:

  Number of true positives

- fp:

  Number of false positives

- prevalence:

  Disease prevalence (optional, for Bayes adjustment)

## Value

PPV value (0-1)
