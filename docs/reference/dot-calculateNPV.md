# Calculate Negative Predictive Value

Calculate Negative Predictive Value

## Usage

``` r
.calculateNPV(tn, fn, prevalence = NULL)
```

## Arguments

- tn:

  Number of true negatives

- fn:

  Number of false negatives

- prevalence:

  Disease prevalence (optional, for Bayes adjustment)

## Value

NPV value (0-1)
