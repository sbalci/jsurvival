# DateTime Converter and Component Extractor

Convert datetime variables to standardized format and extract datetime
components (year, month, day, hour, minute, day name, week number,
quarter, etc.). Features automatic format detection, quality assessment,
and preview of converted data.

## Value

An `R6` class generator object for the `datetimeconverterClass` backend;
used internally by the jamovi analysis wrapper and not called directly.

## Details

This function provides comprehensive datetime conversion capabilities
including:

- Automatic datetime format detection

- Multiple datetime format parsing options

- Quality assessment with min/max/missing statistics

- Preview of converted data

- Component extraction (year, month, day, hour, minute, etc.)

- Day names and month names

- Week numbers, quarters, day of year
