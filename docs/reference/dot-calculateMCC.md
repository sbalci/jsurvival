# Calculate Matthews Correlation Coefficient

Calculate Matthews Correlation Coefficient

## Usage

``` r
.calculateMCC(tp, tn, fp, fn)
```

## Arguments

- tp:

  Number of true positives

- tn:

  Number of true negatives

- fp:

  Number of false positives

- fn:

  Number of false negatives

## Value

MCC value (-1 to 1)
