# melanoma: example dataset for the jsurvival module

Example dataset distributed with the jsurvival jamovi module for
demonstration and testing.

## Usage

``` r
melanoma
```

## Format

A data frame with 205 rows and 7 variables:

- time:

  numeric variable

- status:

  numeric variable

- sex:

  numeric variable

- age:

  numeric variable

- year:

  numeric variable

- thickness:

  numeric variable

- ulcer:

  numeric variable
