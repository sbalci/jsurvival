# Survival Analysis for Continuous Explanatory Variable Documentation

This document provides a comprehensive overview of the Survival Analysis for Continuous Explanatory Variable module, detailing its features, user interface elements, and the underlying R functions.

## Feature Summary

This module performs comprehensive survival analysis for continuous explanatory variables. It includes advanced features such as optimal cut-off determination, multiple cut-offs analysis using various methods, Restricted Mean Survival Time (RMST) analysis, Cox model residual diagnostics, and log-log plots for proportional hazards assessment. It also supports person-time analysis and date-based time calculations, providing robust tools for clinical and epidemiological research.
